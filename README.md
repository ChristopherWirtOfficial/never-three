# Never Three 🎲

An incremental idle dice game where rolling multiples of 3 is dangerous.

## Quick Start

```bash
npm install
npm run dev       # local dev server (Vite)
npm run build     # typecheck + production bundle → dist/
npm run preview   # serve production build locally
```
